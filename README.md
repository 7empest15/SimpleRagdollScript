# SimpleRagdollScript
This is a simple script for ragdoll in FiveM.

FiveM install:

- put the `ragdoll` file in the `resources` folder                                      
- add `start ragdoll` in `server.cfg`                   
- enjoy !                       

Default config:

- default key : U                       
- you can config this script in `client.lua` file<
